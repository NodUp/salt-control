## Getting Started

1. Clone project

```
git clone https://github.com/NodUp/salt-control.git
```

2. Install all dependencies

```
npm i
```

3. Run migrations

```
npx prisma migrate dev --name init
```

4. Run seeds

```
npx prisma db seed
```

5. Run project

```
npm run dev
```

6. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
